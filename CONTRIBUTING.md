# Contributing

Thanks for taking the time to contribute.

## Ways to help

- Report bugs and suggest improvements
- Fix issues or add features
- Improve documentation

## Development setup

1. Fork and clone the repo
2. Open the solution in Visual Studio or use the CLI

```bash
cd Jellyfin.Plugin.InMemoriam

dotnet build -c Release
```

The compiled plugin will be in `bin/Release/net8.0/`.

## Testing

There are no automated tests yet. Please verify changes in a local Jellyfin instance.

## Code style

- Keep changes focused and readable
- Prefer small, well-scoped commits
- Use clear names and avoid unnecessary complexity

## Pull requests

- Describe the problem and the change
- Include steps to verify
- Keep PRs small when possible
